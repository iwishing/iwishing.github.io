Idea的快捷键

环境：
idea：ultimate 2018.1
系统：windows10 家庭版

1.psvm

可以直接输出main方法

2.sout

可以字节输出打印语句
System.out.println();
小技巧：
使用sout的时候，用变量.sout可以直接打印出输出语句，并将变量放在括号中

3.alt+Enter

在需要导入包的变量上按，可以快速导入需要的包

4.ctrl+alt+v

可以快速补全语句，如
new String();
可以快速补全为
String string = new String();

5.alt+insert

可以快速插入构造器，getter和setter方法等

6.ctrl+i

可以快速实现该类所实现的接口的方法

7.ctrl+shift+c

可以快速注释大片代码

8.ctrl+shift+x

可以快速取消大片注释
